# Shop-Here - Интернет-магазин одежды и обуви

Современный интернет-магазин на Next.js 14 с полным циклом от каталога товаров до оплаты заказа. Проект включает публичную часть для покупателей и административную панель для управления контентом.


## Быстрая проверка:
1. Установите зависимости: `npm install`
2. Запустите тесты: `npm run test:run`
3. Проверьте сборку: `npm run build`
4. Запустите dev сервер: `npm run dev`

## Команды для PowerShell:
- Полная проверка: `npm run check`
- Тесты с watch: `npm run test`
- Тесты с UI: `npm run test:ui`
- Покрытие кода: `npm run test:coverage`
- TypeScript проверка: `npx tsc --noEmit`
